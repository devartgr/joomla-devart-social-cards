## 📦 Installation

1. Download latest release ZIP.
2. Go to Joomla Administrator → System → Install Extensions.
3. Upload:

```text
com_devartsocialcards_v1.1.3.zip
```

4. Open:

```text
Components → DevArt Social Cards
```

## 🔄 Updates
